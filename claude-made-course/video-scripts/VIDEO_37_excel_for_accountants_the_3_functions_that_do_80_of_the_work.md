# Video 37 — Excel for Accountants — The 3 Functions That Do 80% of the Work
**Chunk:** 37 · **Module:** Module 07 · **Duration target:** 4 min · **Type:** Talking head / instructor
**Learning objective:** If you can only learn three Excel functions before your first accounting job, learn these: VLOOKUP, SUMIFS, and PivotTables.

## Narration (transcript fallback exists in the in-course `video` chunk)

**0:00 —** If you can only learn three Excel functions before your first accounting job, learn these: VLOOKUP, SUMIFS, and PivotTables. Everything else is optional. These three handle 80% of the real work you will actually do.

**0:30 —** VLOOKUP — or its modern replacement XLOOKUP — is how you match data from two tables. The most common accounting use: a 3,000-row expense report and a separate table of GL account numbers. VLOOKUP pulls the GL number into the right row automatically. Without it, you do that manually for 3,000 rows.

**1:30 —** SUMIFS adds up numbers that meet multiple conditions at once. "Give me total COGS for Tennessee Corridor, in January, from Global CAD Solutions." That's a SUMIFS with three criteria. Syntax: SUMIFS(sum range, criteria range 1, criteria 1, criteria range 2, criteria 2...).

**2:30 —** PivotTables summarize large data sets fastest. Project Code to rows, Vendor Name to columns, Amount to values — in 10 seconds you have a summary that would take an hour of formulas.

**3:30 —** In the Excel lab, raw offshore drafting data — 1,200 rows. Tag project codes with VLOOKUP. Sum by project with SUMIFS. Present with a PivotTable. All three.
